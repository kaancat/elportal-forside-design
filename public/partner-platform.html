Installer tracking script

Indsæt denne kode i <head> sektionen på alle sider af din hjemmeside:

Manuel konvertering (brug hvis din tak-side er dynamisk)

Kald denne funktion i din "succes"-callback — lige efter ordren/kontrakten er bekræftet server-side:

// Kald på bekræftet succes (eksempel) if (window.DinElportal && typeof window.DinElportal.trackConversion === 'function') { window.DinElportal.trackConversion(); }

Ingen persondata — kun anonym click_id. Sørg for at scriptet herover er indlæst i <head> først.

Hvad gør scriptet?

• Fanger automatisk click_id når besøgende kommer fra DinElportal
• Gemmer ID'et sikkert i 90 dage
• Tracker automatisk konverteringer på din tak-side
• 100% GDPR-compliant - ingen persondata

Test URL med click_id

  4. Gå til næste trin og tryk “Tjek Live Status”
